AI Instructions — Page Layout Images (Empty Panels with References)

Goal
Create one layout image per comic page. The image shows only empty panels (white boxes with black borders). Inside each panel, print the panel number and a reference to the future panel image filename. Do not draw any characters, backgrounds, or effects.

Global rendering rules
- Canvas: A4 portrait (2480×3508 px) or 4:5 ratio; safe margin 4% on all sides.
- Background: solid white. Panel borders: #000, 6 px stroke, square corners.
- Inside each panel, centered label in bold black sans-serif:
  - Format: Page.Panel  →  filename.png
  - Example: 2.1  →  page02_panel01.png
- No shadows, no textures, no gradients. Pure blueprint look.
- Export each page as: pageXX_layout.png (e.g., page02_layout.png).

Label format (strict)
- First line: big: P{page}.{panel}
- Second line: small: → page{page padded 2}_panel{panel padded 2}.png
- Example content inside panel 5.3:
  - Line 1: P5.3
  - Line 2: → page05_panel03.png

Panel reference naming
Follow this file scheme for the future panel art:
- page01_panel01.png
- page02_panel01.png, page02_panel02.png, …
- Continue per page/panel.
